# JPMS-DB: 日本私立中学校 包括的基盤データベース

**プロジェクト名**: jp-private-mid-school-db（JPMS-DB）
**現在のフェーズ**: Phase 2進行中（学校実態の大規模並列収集）
**対象**: 日本国内の私立中学校 全785校（令和6年度学校基本調査確定値）
**現状の投入校数**: 525校（関東圏網羅から全国展開へ拡大中）

## 概要

JPMS-DBは、日本国内の私立中学校について、**学校実態**（一次情報ベース）・**教育学術知見**（教育学概念14サブフィールド）・**成長軌道仮説**（OECD/CASEL/PERMA/P21/日本独自指標を統合した成果次元）の三層を統合する包括的データベースです。一次情報の重視・教育学的知見との接続・成長軌道の仮説提示を三本柱に、保護者の学校選び、学校の自己改革、研究者の実証分析、政策決定者の俯瞰のすべてを支える基盤を目指しています。

## 引用倫理ポリシー（要約）

JPMS-DBは一次情報を扱うため、引用倫理に厳格な基準を設けています。

- **学校公式情報**（HP・公式パンフレット・公式SNS）に限定して収録し、口コミサイト・掲示板・個人ブログの直接転載は行わない。
- **未成年情報は原則匿名化**し、生徒個人を特定可能な形では公開しない。
- **rights_status**（公開可・要許諾・要削除）を全レコードで管理し、要許諾分は会員制でのみ提供。
- **削除依頼受付フロー**を整備し、学校・関係者からの申し立てを優先処理する。

詳細は [PUBLIC.md](./PUBLIC.md) を参照してください。
