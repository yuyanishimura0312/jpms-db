2.学校類型 LCA k=8 解析 — 8クラスタごとの特徴と所属校

潜在クラス分析（Latent Class Analysis, k=8）相当の処理として、528校の 学校文化10次元スコアを入力に k-means 法でブラウザ内クラスタリングを実行しました。10次元（自律性／構造度／多様性／学業強度／メンター密度／創造性奨励／競争性／共同体性／国際性／精神性）を z 標準化したのち 8 クラスタに分割し、各クラスタの中心を「型」として読みます。

結果として現れる 8 つの型は、「学術ハイ強度型」「自律探究型」「共同体・精神型」「国際多様型」「競争・実績型」「メンター濃密型」「創造・拡張型」「中庸・標準型」のように、文化的シグネチャの異なる学校群へと分岐します。各クラスタの中心スコア・代表校・所属校数を以下に提示します。

クラスタ中心の文化次元プロファイル

10次元の z 標準化スコアによる。中心からの距離が大きいほど、その次元に対するクラスタの偏倚が強いことを示します。

クラスタ × 学校型（性別×宗教）構成

各クラスタの内部における性別×宗教構成。クラスタが特定の学校型に強く偏倚するかどうかを観察できます。

3.卒業生 alumni_career 領域分布解析 — 学校×領域の散布

JPMS-DB v2 が収録する卒業生キャリア記録は 3,158件。本セクションでは、領域カテゴリ別の総量分布、TOP15 校の活躍量、および「学校×活躍領域」の散布構造を可視化します。卒業生データは 歴史的偉人 DB（GF: 9,178人物）と接続され、各校の長期的な人材輩出傾向を示す主要シグナルとなります。

領域カテゴリ別 総量分布

academic（学術）が圧倒的多数（1,788件）。次いで artist（400）／writer（373）／cultural（285）と続く。

灘・開成・麻布の御三家系が突出。続いて武蔵・慶應普通部・暁星など長い校史を持つ学校群が並ぶ。

学校 × 推定主領域の散布

卒業生輩出量上位15校について、各校の文化次元スコア（自律性 × 学業強度）と推定主領域カテゴリの関係を散布で示します。学術志向（学業強度・高）と創造志向（自律性・高）の二軸で、各校の卒業生輩出パターンを読み解けます。

7.本ページの読み方と限界

方法論メモ

本ページの解析は、v2_data.js（jpms_v2.db フルダンプ・約482KB）のみをデータソースとし、追加のサーバー処理を介さずブラウザ内（Chart.js + 純粋 JavaScript）で算出しています。LCA 類型は厳密な潜在クラス分析ではなく、k-means による近似クラスタリング（k=8、Lloyd 法、ランダム再起動 5 回）を用いた探索的可視化です。Phase F でこれを Mplus / R poLCA による厳密版 LCAに置換する予定です。

時代×成果次元のヒートマップは era_relevance テーブル（50関係）に基づきます。era_relevance は外部知識（FK / MT / CLA / AA / AD）からの理論的予測であり、各成果次元が当該時代でどの程度要請されたか／されるかの「文献的見立て」です。実証的な世代別データではないため、解釈は慎重に行ってください。

公的統計連動セクションの私立中学進学率は、文部科学省「学校基本調査」の各年度公表値を踏まえた 近似参考値です。厳密な分析には文科省 e-Stat の最新版データを直接接続する必要があります。Phase E 後半でこの接続を実装します。

v2 全体としては、個別校の優劣評価ではなく、528校の構造を多層レンズで読み解く知識基盤として設計されています。本ページの可視化はそのレンズ群の最初の出力であり、個別生徒のマッチングや進路選択への直接利用には、Phase F の予測モデル完成と倫理レビューの完了が必要です。
