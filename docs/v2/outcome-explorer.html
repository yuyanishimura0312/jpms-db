Outcome Explorer は、成果次元 (outcome_dim_v2, 77項目 → 目標100+) を中心に置き、それを「学校文化10次元 (Layer 4)」「時代変遷8区分 (Layer 5)」「卒業生キャリア12カテゴリ (Phase E一次集積)」と多面的に接続する独立解析ページです。 各ビューは Chart.js 4.4 を用い、Phase E で蓄積した 528校 × 10次元 × 平均50点規模の文化スコア、528件のalumni実績、50件のera_relevanceデータを統合しています。

本ページの目的は、ある成果クラスタ（例: 認知・学術 / 創造・卓越 / 市場・経営）が、どの学校文化プロファイルを持つ環境で育ちやすいかを「予防的適合モデル」の前段として可視化することにあります。 全体俯瞰 → クラスタ別深堀り → TOP校プロファイル比較 の3段階で読み解く構成にしました。

1. 7大成果クラスタの概観 outcome_dims 77項目を意味で束ねたメタ層

成果次元77項目は、教育心理・教育学・偉人研究・経営学・イノベーション研究・人類学・神話研究の知見を踏まえて、7つの意味クラスタに集約されています。 クラスタは互いに排他的ではなく、特に「認知・学術 × 創造・卓越」「主体性・市民性 × 市場・経営」など隣接領域では重なりを持ちます。 各カードは項目数とサンプル項目を示し、後続セクションで使う共通カラーキーの起点となります。

クラスタ別 成果次元数（横棒チャート）

2. 7成果クラスタ × 学校文化10次元 推奨マトリクス 理論ベースの育成親和性スコア

どの学校文化次元が高い学校が、どの成果クラスタを育てやすいかを示す理論ベースの推奨スコア（0-3）です。 OECD 2030・CASEL・PERMA・偉人研究 (GF) ・経営学 (MG) ・イノベーション理論 (IT) の文献を参照し、各クラスタが必要とする学校環境特性を整理しました。 数値は「最低限必要(1) / 推奨(2) / 強く推奨(3)」の3段階で、空欄は中立を意味します。

0 (中立) 1 (最低限) 2 (推奨) 3 (強く推奨)

たとえば「認知・学術」クラスタは 学業強度・構造度 が高い学校でより育ちやすく、「創造・卓越」クラスタは 創造性奨励・自律性 が、「市場・経営」クラスタは 共同体性・国際性・競争性 が高い学校でより育ちやすい、という仮説的な対応を示しています。 これは Phase G-3 以降の MLM/SEM モデルでの実データ検証対象になります。

3. 時代 × 主要成果次元 era_relevance ヒートマップ 明治〜令和の社会要請変遷

Layer 5 の era_relevance テーブル（50件）から、明治〜令和の各時代における成果次元の重要度（0-10）を抽出し、TOP15項目を縦軸に時代を横軸に並べたヒートマップです。 色が濃いほど当該時代に強く求められた能力です。 時代軸を読むと、明治〜昭和前期は「論理的思考・伝統文化尊重」、平成は「批判的思考・グリット」、令和以降は「創造性・主体性・OECD変革コンピテンシー」が立ち上がる流れが見えます。

本データは過去6時代の実データに基づき、2030s/2050s 区分は CLA (因果階層分析) と FK (フォーサイト基盤) の参照を通じた推定値を含みます。 era_relevance は era-pivot.html (Team G-3) でより詳細に展開されます。

4. 卒業生キャリア12カテゴリ × 7成果クラスタ 対応図 「活躍」を「育成」に翻訳する

卒業生実績データ（alumni_by_category, 計 件）はキャリアカテゴリで分類されています。各カテゴリが、Layer 3 のどの成果クラスタの「結実型」に対応するかを定義しました。 これにより「academic（研究者）が多い学校 = 認知・学術クラスタが結実しやすい学校」という構造的読みが可能になります。

5. 成果次元77項目 フルリスト クラスタ・フレームワーク・予測力で絞り込み可能

全77項目を、cluster / framework / predictability / age_window で絞り込み可能なテーブルです。 predictability は理論的に予測力が高いほど 5 に近く、age_window は測定タイミング（在校中 / 卒業5年 / 10年 / 20年）です。 framework は OECD・CASEL・PERMA・P21・PISA・GF（偉人）・MG（経営）・IT（イノベ）・JP独自・MEXT等の出典を示します。

7. TOP10 alumni排出校 文化プロファイル比較 名門校が共有する文化次元、固有な文化次元

alumni_top_schools 上位10校の学校文化10次元スコアを並べたヒートマップとレーダーチャートです。 多くの名門校に共通する次元（メンター密度・自律性）と、学校ごとに固有な次元（精神性・国際性・競争性）が浮かび上がります。 なお灘・武蔵・桜蔭等で 校は文化スコア未取得（jpms_s_id重複・データ欠損）のため、ここでは取得済みのものを表示しています。
