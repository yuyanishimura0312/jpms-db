# JPMS-DB 公開ポリシー

JPMS-DB（日本私立中学校 包括的基盤データベース）は、学校実態・教育学術知見・成長軌道仮説の三層を統合し、保護者・学校・研究者・政策決定者の意思決定を支えることを目的とする非営利の知識基盤です。本ドキュメントは、データの収集・公開・再利用に関する倫理的取扱いと運用ポリシーを定めるものです。

## 1. 引用倫理ポリシー

### 1.1 一次情報の優先と出典明示

JPMS-DBに収録する全ての記述は、出典を `jpms_sources` テーブル経由で属性化します。各レコードには `source_type`（公式HP・公式パンフ・公式SNS・取材記録・公開報道など）、`primary_or_secondary`（一次／二次）、`reliability_score`、`rights_status`（public / permission_required / removed）を付与します。Research Precision Protocolに基づき、教育学概念の `key_works` には DOI／Semantic Scholar Paper ID／ISBN のいずれかを必須とし、自己申告 `verified` フラグは採用しません。

### 1.2 公開可能ソースの限定

学校に関する記述は、原則として以下の公開ソースに限定します。

- 学校公式ウェブサイト（school_official）
- 学校公式パンフレット・募集要項（official_brochure）
- 学校公式SNS（公式アカウントによる発信）
- 学校が一次主体である報道・公的資料（教育委員会公開文書、e-Stat、学校基本調査等）
- 学校が許諾した取材記録（書面または明示的なメール許諾を保管）

### 1.3 禁止事項

以下の取扱いは明示的に禁止します。

- **口コミサイト・掲示板・個人ブログ・受験匿名コミュニティの直接転載**（要約・統計化・傾向分析を含めて行いません）
- **生徒・保護者の実名・容姿写真等、個人特定情報の収録**（公式HPに本人の許諾下で掲載されたものを除き、JPMS-DB側では再掲しません）
- **未成年の発言の固有名詞付き引用**（学年・性別・属性等の匿名化を徹底）
- **試験成績・内申・健康情報など個別生徒データの保有**

### 1.4 関係者の声（testimonials）

校長・教員・在校生・卒業生・保護者の発言は、`speaker_role` と `rights_level` で厳格に管理します。`rights_level` は `cc-by-nc` / `permission_obtained` / `quote_paraphrase_only` / `internal_only` の4段階で運用し、`internal_only` のレコードは公開ダッシュボード・公開レポートには出力しません。

## 2. 削除依頼・訂正依頼受付フロー

### 2.1 受付窓口

- 受付メール: `jpms-db@example.com`（プレースホルダ。正式アドレスは公開準備中）
- 受付フォーム: 公式ダッシュボード上に設置予定（Phase 2末リリース予定）
- 郵送による申立ても受け付けます（住所はリリース時に開示）

### 2.2 受付対象

以下の主体からの依頼を受け付けます。

- 当該学校（理事長・校長またはその委任を受けた代表者）
- 学校教職員（在籍中／退職後を問わず、本人の発言の取り扱いに関するもの）
- 卒業生・在校生本人（本人発言の取り扱いに関するもの）
- 保護者（自身および未成年の子の発言・情報に関するもの）
- 引用元の著作権者・出版社

### 2.3 処理基準と所要日数

- **緊急対応（72時間以内に非公開化）**: 未成年の個人特定リスク、誹謗中傷、明白な誤情報
- **標準対応（10営業日以内に判断・回答）**: 引用範囲・rights_level の見直し、訂正、追記
- **異議申立て**: 判断結果に異議がある場合、外部有識者を交えた再審議を行います

依頼を受けた段階で当該レコードは即座に `rights_status = removed` に切り替え、調査終了まで公開を停止します（先非公開・後審議の原則）。

## 3. データ更新サイクル

### 3.1 学校実態（school_facts）

- **大規模更新**: 年2回（4月・10月）。学校基本調査・各校の年度切替に合わせて全レコードのレビューを実施。
- **継続更新**: 各校の公式HP更新を月次でクロール監視し、教育課程・募集要項・行事の差分を反映。
- **臨時更新**: 学校の重大な変更（合併・共学化・名称変更）が確認され次第、48時間以内に反映。

### 3.2 教育学概念（concepts / outcomes）

- **半年ごとのレビュー**: 教育学14サブフィールドの key_works を半年ごとに見直し、Semantic ScholarおよびGoogle Scholar Top引用論文を再確認。
- **年次の体系更新**: OECD・CASEL・P21等の枠組み改訂に合わせて成果次元を年次で見直し。

### 3.3 関係者の声（testimonials）

- **個別棚卸し**: 12〜18ヶ月で全件の rights_level を再確認。卒業生発言は卒業後5年で原則非公開化（本人の継続許諾がある場合を除く）。

### 3.4 ダッシュボード・レポート

- ダッシュボード `docs/index.html` は学校実態の更新に同期して自動再生成。
- サンプル12校レポートは半年ごとに改訂版を発行（旧版は版管理し閲覧可能に保つ）。

## 4. 再利用条件

### 4.1 ライセンス

JPMS-DBの公開部分は **CC-BY-NC 4.0**（Creative Commons 表示-非営利 4.0 国際）で提供します。出典として「JPMS-DB / NPO法人ミラツク（<https://yuyanishimura0312.github.io/jpms-db/>）」を明示してください。

### 4.2 別途許諾が必要な利用

- 営利目的での利用（商業出版・有料サービスへの組み込み等）
- データの全件または広範囲なバルクダウンロード・再頒布
- AI学習データセットへの組み込み
- 学校の特定識別情報を含む派生データの公開

これらの利用については `jpms-db@example.com`（プレースホルダ）まで個別にお問い合わせください。

## 5. ガバナンス

- **運営主体**: NPO法人ミラツク
- **倫理審議**: 教育学・法務の外部有識者を含む小委員会を設置予定（Phase 2末）
- **法務監修**: 個人情報保護法・著作権法・学校教育法に関する継続的な監修を取得
- **公開対象**: rights_status = public のレコードのみが公開ダッシュボード／公開レポートに掲載されます
