# 二次情報サイト規約・利用許諾調査報告書（戦略D-1）

**作成日**: 2026-05-05
**調査方法**: WebFetch + WebSearch（自動スクレイピング不使用）

## エグゼクティブサマリー

主要中学受験情報サイト12+を調査した結果、以下の重要結論:

1. スクレイピング自動化は全サイトで禁止または非推奨 — robots.txt と利用規約で暗に禁止
2. 手動引用＋出典明示は教育研究で一定範囲で許容 — ただし掲示板投稿等の個人情報は厳格保護
3. 即座利用可は文科省／日本著作権教育研究会／日能研のみ（A/B評価）
4. インターエデュ／SAPIX／早稲田アカは個別許諾必須（C評価）
5. 学生成績・テストスコア等は一切使用不可

## 倫理スコア評定

| スコア | 定義 | サイト |
|---|---|---|
| A | 明記・引用・出典明示で自由 | 文科省、日本著作権教育研究会 |
| B | 引用ルール遵守＋出典で可 | 日能研、ベネッセ（教育機関向け） |
| C | サイト管理者へ事前問い合わせ | インターエデュ、SAPIX、早稲田アカ、四谷大塚、旺文社、マナビジョン |
| D | DB化・スクレイピング禁止 | 学生成績情報全般、SAPIX 個人成績データ |

## 利用可能TOP3

1. 日本著作権教育研究会 (jcea.info) — 入試問題と著作権の最も信頼できる情報源
2. 文部科学省 (mext.go.jp) — 政府公開情報、自由利用可
3. 日能研 (nichinoken.co.jp) — 著作権ポリシー明示、引用法遵守＋出典明示で基本許可

## JPMS-DB v2 への適用方針

### 即時実行可能
- 文科省統計: 既に school_official_stats に123件投入済み
- 日能研公開情報: 教室・公開テスト情報を出典明示で参考引用

### 個別許諾申請が必要（次フェーズ）
- インターエデュ掲示板投稿の二次利用 → 利用目的・範囲を明示した上で管理者に問い合わせ（通常3営業日応答）
- ベネッセ教育情報 → 教育機関向け申請フォーマットで申請

### 一切利用しない
- 学生成績・テストスコア等の個人情報
- SAPIX等の個人指導記録
- 未成年の個人特定可能な掲示板投稿

## 推奨される次アクション

1. インターエデュ・ベネッセに個別許諾申請フォーマット作成
2. 文科省統計の追加取り込み（学校別データの可能な範囲）
3. 戦略B+C のさらなる精緻化で代替的カバレッジ拡大
