# Benchmark B-1a: 国際大規模教育DBの構造比較

**フェーズ**: B-1a
**実施**: 2026-05-05
**担当**: researcher エージェント

## 1. 主要DB一覧

### NCES Common Core of Data (CCD)
米国公立小中高校130,000校の個票データを毎年集約。非財務（校数・生徒数・人種別）と財務（歳入・歳出）の二層構造。州→教育委員会→学校のヒエラルキー。**個人縦断追跡なし**。

### OECD Education GPS
137カ国超の教育指標プラットフォーム。INES（教育システム指標プログラム）基盤。学習環境・生徒福祉・学業成果を横並び。1-3年周期。**個人レベルコンテキスト変数限定**。

### PISA (OECD)
79-90カ国、15歳生徒60万人、3年周期。認知テスト＋豊富な背景質問票（ESCS、学校風土、家庭背景）。**時点間個人追跡なし（クロスセクショナル）**。

### TIMSS/PIRLS (IEA)
4,000-5,000人/国、複雑なクラスタリング設計（学年→学校→学級→生徒）、多重補完。生徒・家庭・教師・学校・国レベル多層データ。SPSS/SAS/R個票提供。

### UNESCO UIS
200カ国以上、年次。公式統計の国際標準化。**メタデータ・定義の厳密さが強み**。個票非公開。

### What Works Clearinghouse (WWC)
教育介入47,000件超。メタ分析結果ベース、個票なし。サンプル/効果量/信頼区間/エビデンス強度。

### EEF Toolkit
13,000試験から46トピックへ集約。各トピックで「平均効果量・根拠強度・コスト」3段階評価。

## 2. JPMS-DB v2 独自設計の提案

### 階層別データ構造

```
学校レイヤー（個票）
├─ 基本情報（CCD参照）
├─ 構成データ（NCES+UNESCO合致）
├─ 文化・適応指標（PISA参照、深掘り）
│  ├─ 学校風土スコア（教師支援・規律・安全・多様性受容）
│  ├─ 教育理念・特色プログラム（日本独自）
│  └─ 学校組織特性（初任者研修・校務分掌等）
├─ 施策実装データ（EEF+WWC参考）
└─ 縦断追跡フラグ（TIMSS設計参照）

生徒レイヤー（サンプル ≈20% stratified）
├─ 認知評価（読解・算数・理科簡易版）
├─ 背景情報（家庭ESCS・学習習慣・心理社会指標）
└─ 3年継続トラッキング（PISA超越ポイント）

教師レイヤー（学年/教科別）
├─ 経歴・研修履歴
├─ 指導方法
└─ 学級風土

親/保護者レイヤー
├─ 期待度・支援度
└─ 家庭教育環境
```

### 独自性のポイント

| 次元 | NCES | PISA | TIMSS | JPMS v2 |
|---|---|---|---|---|
| 縦断追跡 | なし | なし | なし | ●3年連続 |
| 学校文化深掘り | 限定 | 中 | 中 | ●深掘り |
| 介入施策記録 | なし | なし | なし | ●詳細＋成果 |
| 日本型独自 | × | × | × | ●初任者研修・校務分掌等 |
| 適応度定量化 | × | ESCS | あり | ●多角的指数 |
| 国際比較性 | 米国内 | 国際 | 国際 | ●PISA/TIMSSスーパーセット |

### データ収集周期・対象

- 基本統計: 全校年次（CCD方式）
- 詳細調査: 1,200-1,500校サンプル（地域・規模・設置者で層化）
- 生徒個票: 1,200校 × 20名 = 24,000人/年
- 教師個票: 4,000-5,000人/年
- 親票: 24,000人/年
- 追跡: 同一生徒・学校3年連続

### アクセス方式

1. 個票完全版 → 研究者向け（倫理審査後）
2. 集計版API → 政策立案者・学校法人向け
3. オープンデータセット → 国際学術コミュニティ（3年遅延公開、PISA互換）

## 3. 設計の3要素

JPMS-DB v2 は NCES/PISA/TIMSS との互換性を保ちつつ、3つの独自要素で差異化:

1. **日本型教育文化の定量化** — 学校理念・校務分掌・初任者研修等
2. **3年縦断追跡** — PISA超越
3. **学校施策↔成果マッピング** — EEF/WWC方式
