# 関係者声大幅拡張プロジェクト 実行計画

**現状**: testimonials_v2 = 272件、当初目標 7,000件、達成率 3.9%
**目標**: 第1段階で 1,500件、第2段階で 4,000件、第3段階で 7,000件

## 1. 現状分析

### 主体別内訳
| 主体 | 件数 | 7,000目標時想定 | 不足 |
|---|---|---|---|
| teacher | 110 | 1,584 | -1,474 |
| principal | 70 | 528 | -458 |
| student_current | 49 | 2,640 | -2,591 |
| student_alumni | 18 | 2,640 | -2,622 |
| parent | 25 | 1,056 | -1,031 |
| chairperson | 6 | - | - |

### ボトルネック診断
1. **HP取得校が少ない**（97/528校 = 18%）→ 抽出元が不足
2. **HPに保護者・在校生・卒業生の構造化された声が少ない**（多くの校が PDF への外部リンクのみ）
3. **Wikipedia 卒業生記述は「略歴」のみで「声」ではない**
4. **学校パンフ・進学雑誌・SNS は未活用**

## 2. 追加リサーチ戦略（5本柱）

### 戦略A: HP の深層ページ追加取得
- 取得済 97校 → 528校全件目標
- 優先パス追加: `/voice/`, `/student/voice/`, `/alumni/voice/`, `/interview/`, `/ob_og/`, `/principal_message/`
- 各校あたり最大15-20ページ（現在最大10ページ→拡張）
- **想定追加: +1,000件**

### 戦略B: 取得済HPの精緻再抽出
- raw_html_cache 224ディレクトリ × より精緻な抽出ヒューリスティック
- BeautifulSoup でセマンティックタグ（blockquote、figcaption、article）優先
- インタビュー形式（Q&A）の検出強化
- 複数文の連続抽出（現在は単文中心）
- **想定追加: +500件**

### 戦略C: 取得済HTMLからLLM抽出
- 全raw_html_cache に対してより高度な NLP 抽出
- 「私は」「自分は」「中学時代」等の自己言及キーワード周辺
- 引用符（「」『』""）内テキスト
- **想定追加: +800件**

### 戦略D: 二次情報の倫理的活用
- インターエデュ・みん中・四谷大塚等の保護者口コミ（rights_level=anonymized_only、出典明示）
- 進学雑誌「中学受験パスナビ」「サンデー毎日中学受験」（公開部分のみ）
- 学校説明会動画文字起こし（公式YouTube公開分）
- **想定追加: +1,500件**

### 戦略E: 関連DB横断連動
- 既存ミラツク36DB から学校言及記事抽出
- PE（PESTLE 196,714記事）から私立中言及→学校発信記事ピック
- CI（文化インテリジェンス 576,434記事）から教育文化言及
- UPR（大学プレスリリース）の高校・中高一貫言及
- **想定追加: +1,000件**

合計想定: **+4,800件 → 5,072件**（目標7,000まで2,000件不足、戦略Bと戦略Cの精緻化で補完）

## 3. 実行体制

### Round X-1（即時実行・3並列）
- T-A: HP取得拡張（残り431校、4-5並列で全件取得）
- T-B: raw_html_cache 精緻再抽出（既存224 dir）
- T-D: 二次情報倫理的活用（インターエデュ・パスナビの公開部分）

### Round X-2（次サイクル）
- T-C: LLM 抽出パイプライン
- T-E: 関連DB横断連動

## 4. 倫理規律（最重要）

- robots.txt 厳守（特に二次情報サイトへのアクセス）
- 5秒/req遅延
- 引用は短文 + 出典明示（rights_level=quoted_with_attribution）
- **未成年の個人特定情報は完全匿名化**（speaker_attribute は「中学生」「保護者」等の generic）
- 削除依頼は48時間以内対応
- インターエデュ等の口コミサイトは規約確認、引用許可範囲のみ
