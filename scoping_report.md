# JPMS-DB Phase 0 スコーピング統合報告書

**プロジェクト名**: 日本私立中学校 包括的基盤データベース（jp-private-mid-school-db、略称 JPMS-DB）
**フェーズ**: Phase 0 — Scoping Research（リサーチ計画を立てるためのリサーチ）
**実施日**: 2026年5月4日〜5日
**作成者**: 西村勇也／Claude（academic-db-builder）／10並列リサーチワーカー
**対象**: 日本国内の私立中学校 全785校（令和6年度学校基本調査確定値）

## エグゼクティブサマリー

本プロジェクトは、日本国内の私立中学校785校について、学校実態・教育学術知見・成長軌道仮説の三層を統合する包括的データベース「JPMS-DB」を構築するものである。本プロジェクトの最大の特徴は、既存の学校情報集積サービスとは根本的に異なり、第一に一次情報を体系的に収集すること、第二に教育学・発達心理学・学習科学・SEL等の学術的知見と接続すること、第三に「どのような子どもがどの学校でどのように成長しうるか」を仮説として提示できる構造を持つことである。

Phase 0スコーピングの結果、私立中学校の総数（785校）、地域分布（東京都187校・全体の24%）、男女別構成（共学84%）、宗教系列分布の概観が確定した。既存の集積サービス（みんなの中学校情報、インターエデュ等）と書籍系（声の教育社、晶文社等）、塾系偏差値表（SAPIX、四谷大塚、日能研）の構造とカバー範囲も明らかになった。教育学術領域では、国内の主要11学会・J-STAGE主要誌、海外のAERA・OECD・CASEL等の組織と中核理論群、エビデンスベース教育リソース（EEF、WWC、Hattie）の体系が整理された。

国際的な学校情報DBの先行事例（GreatSchools、Niche、SchoolDigger、GOV.UK Compare School Performance、Good Schools Guide、IBO World Schools等）の比較分析を通じて、JPMS-DBが既存にない「三層統合×一次情報×日本的価値観」という独自空間を確保できることが確認された。データスキーマ初版（v0）として、層1の学校実態（schools・curriculum・stats・outcomes・admissions・facilities・evaluations）、関係者発言（testimonials）、層2の教育学概念（concepts・concept_relations）、層3の成果次元と成長軌道仮説（outcome_dimensions・growth_hypotheses・school_concept_links・school_outcome_weights）、共通レイヤ（sources・fetch_jobs）からなる15テーブル構成のSQLite設計を完成させた。倫理・rights方針として、CC-BY-NC + 一部会員制のハイブリッド公開モデル、未成年情報の原則匿名化、削除依頼受付フローを定めた。

Phase 1への提言として、(a) 全785校の学校HP一次情報の体系的取得、(b) 教育学術概念500-800件の文献ベース収集、(c) 関係者発言の倫理パイロット運用、(d) 成果次元20-30個の確定、(e) 法務監修と私学団体への通知、を並行的に開始することを推奨する。

## 1. プロジェクトの背景と意義

私立中学校は日本の中等教育において独自の教育理念・カリキュラム・関係者文化を形成しており、その多様性は公立学校とは比較にならないほど豊富である。しかし、保護者・教育関係者・研究者がこの多様性を理解しようとする際、現状の情報源には深刻な制約がある。集積サイトはユーザー生成コンテンツに依存し情報の信頼性が不均一であり、書籍は地域偏在と更新遅延がある。塾系偏差値表は学力ランキングに矮小化され、各校の教育哲学や関係者の生の声を構造的に把握する手段は事実上存在しない。

加えて、教育学・発達心理学・学習科学の最新の学術知見は、学校選びの実践と切り離されている。OECD Learning Compass 2030、CASEL SEL Framework、Heckmanの非認知能力研究、Dweckの成長マインドセット、Steinbergの思春期脳発達理論など、子どもの教育にとって本質的に重要な国際的知見は学術界に閉じており、保護者と学校の意思決定にはほぼ届いていない。

JPMS-DBは、この二つの欠落を同時に解決する三層統合データベースとして構想される。すなわち、第一の層で学校実態を一次情報重視で構造化し、第二の層で教育学術知見を体系化し、第三の層で「どのような子どもがどの学校でどのように成長しうるか」の仮説を、学校特性と学術知見の交差点として記述する。これにより、保護者は学校選びの根拠を得、学校は自己改革の参照点を得、研究者は学校×成果の実証研究基盤を得、教育政策決定者は私学セクター全体の俯瞰を得ることができる。

## 2. 私立中学校の総数と分布（W01-03）

文部科学省が2024年12月18日に公表した令和6年度学校基本調査の確定値によれば、日本の私立中学校の全国総数は **785校** である。これは令和3年度の778校から3年間で7校増加した数値であり、長期的には共学化と中高一貫化が進む中で総数は緩やかな増加傾向にある。

地域分布は極めて偏在している。東京都が187校（全国の24%）と圧倒的に多く、神奈川県の63校（8.1%）、大阪府の61校（7.8%）が続く。三大都市圏への集中が著しく、一方で山形県には私立中学校が一校も存在しない。この地域偏在は、JPMS-DBの収集パイプライン設計に重要な含意を持つ。首都圏・関西圏では収集対象校が稠密で詳細情報が豊富である一方、地方校では情報密度そのものが薄いため、収集戦略を地域別に最適化する必要がある。

男女別構成では共学化が劇的に進行している。1980年時点で男子校425校・女子校724校であったのに対し、現在は男子校約92校（12%）・女子校約266校（34%）・共学校約662校（84%、複数集計の重複あり）となっている。東京都に限定すると、男子校31校（16.9%）・女子校65校（35.5%）・共学校87校（47.5%）と全国平均より男女別学比率が高い。中高一貫教育校は2023年時点で全国678校（公立含む）と推定されているが、私立785校のうち完全型・併設型・連携型がそれぞれ何校かは公表統計に明示がなく、Phase 1での詳細調査が必要である。

宗教・思想系列の分類は公的統計が存在せず、宗教情報リサーチセンター等の二次情報によれば、キリスト教系（カトリック・プロテスタント・聖公会）が全体の20-25%、仏教系が15-20%、神道系が5%以下、残りが無宗教の建学の精神型と推定される。Phase 1で各宗教団体・私学協会への直接照会により正確な分類を確立する必要がある。

私学団体としては、日本私立中学高等学校連合会（中高連、千代田区九段北）が47都道府県の私学協会を統括する中心組織であり、年1回の実態調査を実施している。東京私立中学高等学校協会（419校加盟）、神奈川県私立中学高等学校協会、大阪私立中学校高等学校連合会等の主要県協会は独自の統計・要覧を発行している。これらは Phase 1での一次情報収集の正規ルートとして極めて重要である。

## 3. 既存情報集積サービスのカバー範囲と限界（W04-06）

日本国内で私立中学校情報を集積する既存のサービスは多層的に存在するが、いずれもJPMS-DBが目指す統合性には到達していない。

Web集積サイトでは、**みんなの中学校情報**（10,285校・口コミ131,195件）が日本最大規模を誇り、シリタス（1,085校）の約10倍のデータ量を持つ。インターエデュ・ドットコムは月間PV約1億の掲示板型サービスとして高い影響力を持つが、構造化データではない。スタディ（中学受験スタディ、関東370校）は声の教育社系列の地域特化型である。これらに共通するのは、ユーザー生成コンテンツに依存することと、教員・職員の人事データ、歴史的文脈、進学実績の時系列、社会経済的背景といった重要次元が体系的に欠落していることである。

書籍系では、声の教育社「中学受験案内」、晶文社「首都圏中学受験案内」、大学通信「中高一貫校データブック」、学研「中学校受験案内」、旺文社「中学受験ガイド」、進学レーダー（みくに出版）が主要シリーズであり、特に晶文社シリーズは60年の蓄積を持つ。書籍版は「学費・カリキュラム・教育方針の深い記述」で集積サイトを上回るが、年次更新の遅延と地域偏在（首都圏中心）が制約となる。

塾系では、四谷大塚80偏差値、SAPIX偏差値、日能研R4偏差値が業界標準として参照されている。これら三者は同一校でも異なる値を提示するため、JPMS-DBでは並存させて記録する設計（jpms_school_stats の deviation_source 区別）が必要である。

既存ソースに共通する5つの構造的欠落は次の通りである。第一に、教員・職員の人的資本データ。第二に、歴史的文脈・経営系譜。第三に、進学実績の時系列化。第四に、社会経済的背景の統計。第五に、複合的学校分類軸（偏差値以外の軸での類型化）。これら5つの欠落こそ、JPMS-DBが埋めるべき差別化空間である。

Phase 1での優先ソースは、第一にみんなの中学校情報（マスターリスト+口コミ自然言語処理）、第二に晶文社「首都圏中学受験案内」（標準粒度の基準）、第三にシリタス（全国対応web）、第四に塾系偏差値表（多層的難易度指標）、第五にスクマ・metasc（リアルタイム情報の補完層）の順に位置づける。

## 4. 一次情報の入手経路マップ（W07／W08-09）

JPMS-DBの最大の差別化要因は、学校HP・パンフ・関係者発言という一次情報を体系的に収集することである。Phase 0では、このための入手経路を実地サンプル調査をもとにマッピングした。

学校HPの構造分析では、典型的なセクション構成として、建学の精神、教育方針、カリキュラム、探究・特色教育、学校生活、入試情報、進路実績、国際教育、施設、保護者向け情報、お知らせの11セクションが共通項目として観察された。一方で各校の独自項目として、教員コラム、卒業生インタビュー、校長メッセージ、在校生の声、学校評価レポート、研究紀要、SDGs取り組み、ICT活用報告などが多様な形態で存在する。データ取得の難易度は校によって異なり、HTMLで構造化されている学校（取得容易）から、PDFのみで提供される学校（OCR必要）、JS動的生成（ヘッドレスブラウザ必要）まで分布する。SNS連携状況も急速に進展しており、公式X、Instagram、YouTubeチャンネルを持つ学校が増えている。

SNS・ブログ・動画系の一次情報経路は多様で広大である。保護者ブログ・受験記録ブログはアメブロ・はてなブログ・noteを中心に膨大な量が存在し、中学受験コミュニティを形成している。X（Twitter）には主要な受験親・塾講師・私立中学関係者のアカウントがあり、ハッシュタグ運用（#中学受験、#〇〇中学校）が活発である。Instagramでは公式アカウントを運用する学校が増えつつあるが、在校生の私的アカウントが学校文化を伺わせる重要なソースともなる。YouTubeでは受験系YouTuber（コベツバ、ジュクコ、にしむら先生等）が学校レビュー動画を制作しており、学校公式YouTubeチャンネルや学校説明会動画も整備が進む。5chの中学受験板・各校スレッドは情報源としては存在するが、信頼性とライセンス上の制約が大きく、本DBでは「言及の存在のみ記録、転載しない」方針を取る。

これらの経路にはそれぞれ固有の倫理的制約があり、Phase 1の収集設計では、収集の合法性・規約遵守・未成年保護を担保した上で、商用APIの正規利用または手動収集を中心とする。スクレイピングは技術的に可能であっても規約違反になりうる領域は避ける。

## 5. 教育学術知見の体系（W10／W11-12）

JPMS-DBの第二層を構成する教育学術知見について、国内・海外の主要な学会・ジャーナル・理論を体系化した。

国内では、日本教育学会（1941年設立、3,000人規模）、日本教育心理学会、日本教育社会学会、日本カリキュラム学会、日本教育方法学会、日本発達心理学会、日本特別ニーズ教育学会、日本教育工学会、日本道徳教育学会、日本生徒指導学会、日本キャリア教育学会の主要11学会が体系の柱となる。これらの機関誌はJ-STAGEを通じてオープンアクセスで配信されており、年間2,000件以上の査読論文が中等教育・思春期発達領域に蓄積されている。研究機関としては、国立教育政策研究所（NIER）、国立特別支援教育総合研究所、国立青少年教育振興機構、ベネッセ教育総合研究所、河合塾教育研究開発本部が中核である。中等教育・思春期発達領域の主要研究者として、苅谷剛彦・本田由紀・耳塚寛明・志水宏吉・舞田敏彦（教育社会学）、苫野一徳・佐藤学・奈須正裕（教育学・カリキュラム）、遠藤利彦・無藤隆（発達心理学）、速水敏彦・篠ヶ谷圭太郎（学習動機・自己調整学習）の名が上がる。

海外の最先端研究では、AERA（米国教育研究学会、24,000名）、EERA（欧州教育研究学会）、IEA（TIMSS/PIRLS主催）、OECD CERI（PISA主催）、UNESCO MGIEP、Brookings Institution、Jacobs Foundationの7大組織が体系の核となる。主要ジャーナルとしてReview of Educational Research（IF 7.4）、Journal of Educational Psychology（IF 6.4）、American Educational Research Journal、Child Development、Learning and Instruction（IF 4.7）、Mind, Brain, and Educationが中心軸である。

理論体系としては、学習科学領域でNational Academies "How People Learn II"（2018）、Sweller認知負荷理論、Mayerマルチメディア学習理論、Zimmerman自己調整学習が基盤である。非認知能力ではHeckman、Dweck（成長マインドセット）、Duckworth（GRIT）、OECD Skills for Social Progressが中核となる。SEL領域ではCASEL 5コンピテンシーフレームワークと、Durlak et al.（2011、270,034名）、Taylor et al.（2017、97,406名）の二大メタ分析が必須参照文献である。思春期発達ではErikson同一性形成、Marcia同一性地位理論、Steinberg Dual Systems Model、Self-Determination Theory（Deci & Ryan）、キャリア発達ではSuper／Savickas、ウェルビーイングではSeligman PERMAモデルとOECD Happy Life Dashboardが主軸となる。エビデンスベース教育としてはEEF Toolkit（英国、2,950研究）、What Works Clearinghouse（米国IES）、Hattie Visible Learning（815メタ分析）が政策実践の参照点として機能している。

これらの知見は、層2の jpms_education_concepts テーブルにサブフィールド分類（learning_science / motivation_dev / noncognitive / sel / adolescent_dev / career_dev / wellbeing / evidence_based / comparative / japanese_pedagogy / curriculum / assessment / inclusion / sociology_of_ed の14カテゴリ）で構造化される。Phase 1では各サブフィールドあたり50-100件、合計500-800件の文献ベース概念収集を目標とする。

## 6. 国際先行学校情報DBの分析と差別化（W13-14）

国際的な学校情報DBの先行事例分析により、JPMS-DBの独自性が明確になった。

米国のGreatSchools.orgは全米約150,000校を網羅する最大規模で、Summary Rating（1-10）とEquity Ratingの二軸評価、API/データフィード自動化が強み。Niche.comは300万件のレビューと大学進学検証データを結合した複合ランキング型、SchoolDiggerはAPI駆動型の正規化フィード配布、U.S. News & World ReportはAP/IB履修率中心のランキングメディアである。

英国のGOV.UK Compare School Performance Serviceは政府公的データのオープン化を徹底し、商業的配慮を排除した透明性重視型である。Good Schools Guideは独立系教育ジャーナリズムの伝統を持ち、UK内2,500校の私立学校に対して編集部訪問取材で「教育文化」を文章化する。これは日本のJPMS-DBが目指す一次情報重視と最も近い哲学を共有する事例である。

欧州ではSchulen.de（独）、アジアではSchoolAdvisor.my（マレーシア）、SchoolPro（シンガポール）等が国別に存在するが、私立中等教育に特化した日本型の集積はほぼ存在しない。国際バカロレア系ではIBO公式の学校検索とInternational Schools Databaseが認定校追跡の一次ソースとなる。

これらと比較した上で、JPMS-DBの差別化空間は次の10点である。第一に、実態×学術×成長軌道の三層統合アーキテクチャ。第二に、一次情報の徹底化（学校HP、公的統計、関係者発言の構造化収集）。第三に、保護者・在校生・卒業生・教員という多角的関係者の発言の体系的取得。第四に、教育学知見との双方向接続（実態→概念のForward mappingと、学術問い→学校のReverse mapping）。第五に、SEL・非認知スキルの体系的追跡。第六に、進路・人生成果との長期連携想定。第七に、社会的自立・地域貢献・道徳性・集団協調性といった日本的価値観の埋め込み。第八に、学校間差異の構造的説明（理念→カリキュラム→実践→成果の因果記述）。第九に、学校が情報開示レベルを選択可能なプライバシー・信頼設計。第十に、Open Dataと学校主体性のバランスである。これらは既存のいずれの国際DBにも存在しない統合空間であり、JPMS-DBの存在意義を明確に画定する。

## 7. 教育成果指標と層3の理論基盤（W15-16）

層3「成果仮説層」の理論基盤として、国際的な成果指標体系を網羅的に整理した。

OECD系では、Learning Compass 2030がコンピテンシー（知識・スキル・態度・価値観）の包括的枠組みを提示し、Future of Education and Skills 2030プロジェクトとともに21世紀の教育目標を定義する。PISAは読解・数学・科学リテラシーに加え、Innovative Domain（協調的問題解決、グローバルコンピテンシー、創造性）を追加し、TIMSS／PIRLS／TALIS／PIAACが補完体系を構成する。CASEL SEL Frameworkの5コアコンピテンシー（Self-Awareness、Self-Management、Social Awareness、Relationship Skills、Responsible Decision-Making）は社会情動的領域の標準フレームワークである。21st Century Skillsフレームワークでは、P21 Partnership for 21st Century Learning、ATC21S（Griffin & Care）、World Economic Forum、Brookings "Skills for a Changing World" が複合的な指標体系を提供する。

キャリア発達と成人期成果では、Hauser & Warren／Heckman の長期追跡研究、Career Construction Theory（Savickas）、教育の長期効果研究（earnings、health、civic engagement）が層3の参照軸となる。ウェルビーイング領域ではOECD Better Life Index、Seligman PERMA-H、Children's Worlds Survey、こども家庭庁の子どもウェルビーイング指標が日本での適用可能性を持つ。創造性・批判的思考の評価としてはTorrance Tests、California Critical Thinking Skills Test、PISA Creative Thinking 2022、市民性ではICCSが標準である。

これらを統合し、JPMS-DBの jpms_outcome_dimensions テーブルには20-30個の成果次元を実装する予定である。各次元はframework（OECD_LC2030 / CASEL / P21 / PERMA / PISA_WB / japanese_independent等）でタグ付けされ、measurability（quantitative / qualitative / mixed / conceptual）と relevance_age（12-15、12-22等）で属性付与される。これにより層1の学校特性と層3の成果次元の関連付け（jpms_school_outcome_weights）と、層2の教育学概念と層3の成果次元の理論的接続が構造化される。

## 8. データスキーマ設計（W17-18）

JPMS-DBのスキーマv0は SQLite を基盤として、15のメインテーブル＋3つのFTS5仮想テーブルから構成される。詳細は schema_v0.sql ファイルに収録した。

層1の学校実態層は、jpms_schools（マスタ）を中心に jpms_school_curriculum（特性）、jpms_school_stats（年度統計）、jpms_school_outcomes（進路実績）、jpms_school_admissions（入試）、jpms_school_facilities（施設）、jpms_school_evaluations（評価）の7テーブルで構成される。学校マスタにはreligious_affiliation、gender_type、integrated_typeのCHECK制約付き分類軸を持たせ、Phase 1での収集を構造化する。

関係者発言テーブル jpms_testimonials は本プロジェクトの核心であり、speaker_category（在校生・卒業生・保護者・教員等の8分類）、medium（学校HP・パンフ・SNS・書籍等の13分類）、rights_level（public・quoted_with_attribution・anonymized_only・permission_required・withhold の5段階）を厳格に管理する。jpms_testimonial_concept_links により発言と教育学概念のタグ付けを可能とする。

層2の教育学術層は、jpms_education_concepts と jpms_education_concept_relations の2テーブルで構成される。これは経営学DB（mg）・イノベーションDB（it）の構造を踏襲しており、サブフィールド14カテゴリ、relation_type 10種類を採用する。

層3の成果仮説層は、jpms_outcome_dimensions（成果次元マスタ）、jpms_growth_hypotheses（成長軌道仮説）、jpms_school_concept_links（学校→概念のリンク）、jpms_school_outcome_weights（学校→成果次元の重み）の4テーブルで構成される。成長軌道仮説は短期（中卒時）・中期（大卒時）・長期（30代）の三時点で記述し、参照する概念IDと成果次元IDをJSON配列で保持する。

共通レイヤとして jpms_sources（出典マスタ、source_typeとrights_status・primary_or_secondaryで属性化）、jpms_fetch_jobs（取得ジョブ追跡）の2テーブルが全体を支える。すべての主要テーブルから source_id 参照を持ち、出典なしでのデータ投入を技術的に不可能とする設計である。

FTS5全文検索は jpms_testimonials_fts、jpms_concepts_fts、jpms_schools_fts の3つで、Phase 2以降の高速検索を支える。

スキーマv0確定に向けてPhase 1初頭で解決すべき残課題として、生徒数・教員数の年度時系列と現在値の二重持ちの解消、学校間関係（系列校・姉妹校・合併）テーブルの追加、入試問題本体・教材サンプルの大容量バイナリ取扱い、本DBと既存DB（mg、it、academic）とのcross-domain関係テーブル追加判断、curriculumフラグの正規化方針の最終決定の5項目がある。

## 9. 倫理・rights・引用ポリシー（W19）

一次情報重視のJPMS-DBにおいて、引用・出典・rightsの管理は基盤的に重要である。Phase 0では包括的な倫理ポリシーを整備した。

著作権法第32条の引用要件（公表・必然性・明瞭区別性・主従関係・出所明示）を踏まえ、本DBでは引用テキストには必ずJPMS-DB側の分析・タグ付け・関係付けを伴わせ、単なる転載集にならない構造を保証する。入試問題は「出題傾向の文章記述」のみを扱い、問題本体は転載しない。

個人情報保護では、未成年（中学生本人）の発言・氏名・写真等は学校HP掲載のものでも本DBでは原則匿名化（学年と学校名のみ保持）とし、要配慮個人情報に近い扱いとする。卒業生は本人の氏名公表同意が判断可能なケースのみ実名保持、保護者のSNS発言は原則匿名化、教員・校長は職務として公表されている発言を実名で扱える。

各情報源カテゴリ別の引用ルールとして、学校公式HP・パンフは出所明示で引用可、商用書籍・私学ガイドは「該当校に記載がある事実のメタデータ」のみ記録、口コミサイトは投稿の傾向分析のみ、SNS（X・Instagram・TikTok・YouTube・note・ブログ）は商用APIの正規利用に限定し原文長文転載は行わない。5chは規約上の制約から「言及の存在のみ記録、転載しない」方針とする。

公開モデルとしては、CC-BY-NC（非商用公開）を基本とし、testimonialsの一部は会員のみ閲覧とするハイブリッド運用を推奨する。学校・関係者からの異議申立て・削除依頼への対応フロー（48時間以内の暫定非表示化、事実確認後の恒久対応、削除ログ保存）を整備する。

Phase 1着手前に、教育・著作権分野の法務専門家による監修、サンプル10校での倫理パイロット運用、私学団体（中高連、各都道府県協会）への情報共有と通知文書テンプレート整備、CC-BY-NC + 会員制ハイブリッドの公開モデル最終決定を実施する必要がある。

## 10. 想定規模とコスト・期間試算

Phase 1以降のスケール感を整理する。

層1（学校実態）は対象785校×平均30フィールド（マスタ＋付随テーブル）で約24,000レコード相当。年度統計・進路実績・入試の時系列はPhase 1で過去5年分を遡れば追加で78,500レコード（785校×5年×20指標）規模となる。

層1関連の関係者発言（testimonials）は、各校あたり保護者・在校生・卒業生・教員・第三者の5カテゴリ×年5件×3年分を目標とすると、785校×75件 = 約58,875件規模。学校HP掲載分を中心に最初の1年で20,000件程度の収集が現実的である。

層2（教育学概念）は、サブフィールド14カテゴリ×平均50件 = 700件を目標。各概念の関係性は概念数×3-5倍 = 2,100-3,500件。これはmg DB（3,369概念）・it DB（9,839概念）と同水準の精緻さを持つ。

層3（成果仮説）は、成果次元20-30個、成長軌道仮説は学校ごとに1-3個 = 約1,000-2,300仮説、school_concept_linksは785校×平均5概念 = 約3,925件、school_outcome_weightsは785校×平均10次元 = 約7,850件規模。

共通レイヤのsourcesは全レコードの出典として、推定50,000-80,000件のソースエントリが集積される。

期間としては、Phase 1（情報収集設計と倫理パイロット）に2-3カ月、Phase 2（大規模並列収集）に4-6カ月、Phase 3（品質検証）に1-2カ月、Phase 4（関係ネットワーク構築）に1-2カ月、Phase 5（成果物生成）に1-2カ月、合計で12-18カ月を想定する。Codex/Claude並列ワーカー（最大10並列）を活用しつつ、人的なフィールド取材・法務監修・学校通知が並走する複合プロジェクトとなる。

## 11. Phase 1への提言と次のアクション

Phase 0スコーピングの結果を踏まえ、Phase 1で着手すべきアクションを優先度順に整理する。

最優先で着手すべき項目として、第一にスキーマv0の実SQL適用とサンプル10校の試験投入、第二に層1の必須情報チェックリスト確定（W01-03の公的統計データのe-Stat実ダウンロードを含む）、第三に教育学術概念の必須リスト確定（W11-12のチェックリスト23本を起点）、第四に成果次元20-30個の確定とOECD/CASEL/P21等のframework属性付与、第五に法務監修と私学団体への通知準備が挙げられる。

並行して進めるべき項目として、収集パイプラインの設計（学校HP収集の標準フィールド設計、SNS取得の規約遵守設計、書籍OCR処理パイプライン）、倫理パイロット運用（10校×全領域の試験収集と削除依頼受付フロー検証）、CC-BY-NC + 会員制ハイブリッドの公開モデル最終決定、Codex/Claude並列収集ワーカーのテンプレート整備が必要である。

Phase 1完了の判定基準として、サンプル10校で全15テーブルが矛盾なく埋まり、testimonialsが各校5件以上収録され、教育学概念が100件以上投入され、成果次元20個以上が確定し、ethics_logと監査ログが運用されている状態を目指す。これにより Phase 2（全785校への大規模展開）への移行条件が整う。
